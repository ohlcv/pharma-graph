# 药学知识图谱 — 产品文档

## 1. 产品概述

**药学知识图谱**（pharma-graph）是一个基于知识图谱可视化技术，帮助药学学习者整理、关联和记忆药学专业知识的工具。

核心思路：用 Markdown 写笔记，用图谱看知识。

---

## 2. 核心概念

### 2.1 节点（Node）

每个药学概念、药物名称、术语、方法等，都对应图谱中的一个**节点**。

节点的元数据定义在 Markdown 文件的 frontmatter 里：

```yaml
---
data:
  id:          # 唯一标识，英文/拼音，全局唯一
  label:       # 显示名称，中文
  type:        # 类型，影响节点形状
  category:    # 分类，影响节点颜色

location:      # 位置信息（所属教材章节）
  book:        # 书名
  part:        # 篇
  chapter:     # 章
  section:     # 节
  point:       # 考点编号
  item:        # 要点
  subsection:  # 子节（最小层级）

tags:          # 自由标签列表

summary:
  short:       # 一句话定义（悬停显示）
  full:        # 详细解释（点击弹出）

edges_out:    # 从本节点出发的所有边（列表）
  - target:   # 目标节点id
    type:     # 关系类型
    reason:   # 为什么我要指向它
---
正文：听课笔记...
```

### 2.2 边（Edge）

边代表两个节点之间的关系。

- **source**：起始节点（即当前 Markdown 文件对应的节点）
- **target**：目标节点（由 `edges_out[].target` 指定）
- **type**：关系类型
- **reason**：这条边为什么存在（可选）

只需要在发起关系的节点里写 `edges_out`，不需要两边都写。

---

## 3. 关系类型（8种）

| 关系 | 含义 | 数学属性 | 示例 |
|------|------|----------|------|
| `isa` | 属于 | 反对称 + 传递 | 巴比妥类 isa 镇静催眠药 |
| `partOf` | 组成部分 | 反对称 + 传递 | GABA受体 partOf 中枢神经系统 |
| `related` | 相关 | 对称 | 阿司匹林 related 华法林 |
| `prereq` | 前置依赖 | 反对称 | 中枢肌松药 prereq 大脑和脊髓 |
| `causes` | 导致 | 非对称 | 巴比妥类 causes 宿醉效应 |
| `treats` | 治疗 | 非对称 | 苯二氮卓类 treats 失眠 |
| `similar` | 类似 | 对称 | 巴比妥类 similar 苯二氮卓类 |
| `mechanism` | 作用机制 | 非对称 | 咖啡因 mechanism 腺苷受体 |

### 3.1 边的视觉区分

每种关系类型在图谱中通过**颜色 + 线型 + 箭头形状**三维度区分：

| 关系 | 颜色 | 线型 | 箭头 |
|------|------|------|------|
| `isa` | 蓝色 `#60a5fa` | 实线 | 三角箭头 |
| `partOf` | 青色 `#22d3ee` | 实线 | 三角箭头 |
| `related` | 灰色 `#94a3b8` | 虚线 | 圆点箭头 |
| `prereq` | 橙色 `#fb923c` | 点线 | 菱形箭头 |
| `causes` | 红色 `#f87171` | 虚线 | T形箭头 |
| `treats` | 绿色 `#34d399` | 实线 | 三角箭头 |
| `similar` | 黄色 `#fbbf24` | 虚线 | 圆点箭头 |
| `mechanism` | 紫色 `#a78bfa` | 实线 | 三角箭头 |

## 4. 节点类型（影响形状）

| 类型 | 形状 | 适用场景 |
|------|------|----------|
| 概念（concept） | 圆角矩形 | 定义、理论 |
| 药物（drug） | 椭圆 | 药物名称 |
| 术语（term） | 矩形 | 专业术语 |
| 方法（method） | 六边形 | 检测/制备方法 |
| 疾病（disease） | 菱形 | 疾病名称 |
| 靶点（target） | 桶形 | 受体、酶、离子通道 |

---

## 5. 工作流程

```
编写 Markdown 笔记
        ↓
frontmatter 定义节点元数据
edges_out 定义关系
        ↓
程序扫描所有 .md 文件
        ↓
构建节点 + 边数据
        ↓
Cytoscape.js 渲染知识图谱
        ↓
交互探索：点击、缩放、搜索、布局切换
```

---

## 6. 使用场景

### 场景一：备考复习

将教材考点写成卡片笔记，通过图谱直观看到知识点之间的关联，找出自己的薄弱环节。

### 场景二：知识整理

听完课后整理笔记，梳理概念之间的 isa/partOf 关系，加深理解。

### 场景三：考前冲刺

通过图谱快速浏览全章知识结构，聚焦高频考点。

## 7. 术语表

| 术语 | 说明 |
|------|------|
| frontmatter | Markdown 文件顶部的 YAML 元数据区域 |
| node（节点） | 图谱中的圆点，代表一个知识点 |
| edge（边） | 图谱中的连接线，代表两个知识点之间的关系 |
| layout（布局） | 节点在画布上的排列方式（环形、树状、力导引等） |
| highlight（高亮） | 选中节点后，与它关联的节点/边高亮显示 |
| selector（选择器） | Cytoscape.js 中用于匹配节点/边的条件表达式 |
| stylesheet（样式表） | 定义节点和边外观的规则集合 |
