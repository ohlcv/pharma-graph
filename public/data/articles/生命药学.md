# 生命药学

## 概述

生命药学（Life Sciences for Pharmacy）是药学教育的基石，提供**人体解剖、生理、病理**的基础知识，是理解疾病机制和药物作用靶点的起点。

---

## 核心内容模块

### 解剖学基础

- **神经系统**：大脑皮层分区、脑室结构、血脑屏障（BBB）
- **心血管系统**：心脏传导系统、血管分布（动脉/静脉/毛细血管）
- **呼吸系统**：气道结构（上下呼吸道）、肺泡-毛细血管气体交换界面
- **消化系统**：胃肠道分段（食管→胃→小肠→大肠）、肝脏解剖
- **内分泌系统**：下丘脑-垂体-靶腺轴（HPA axis）

### 生理学基础

| 系统 | 关键生理过程 | 临床关联 |
|---|---|---|
| 神经 | 动作电位传导、突触传递 | 全麻药、镇痛药机制 |
| 心血管 | 心脏泵血、血压调节（RAAS） | 降压药、抗心衰药 |
| 呼吸 | 气体交换、通气/血流比 | 平喘药、COPD治疗 |
| 消化 | 胃酸分泌、蠕动调节 | 抑酸药、促动力药 |
| 内分泌 | 激素负反馈调节 | 糖尿病用药、甲状腺药 |
| 泌尿 | 肾小球滤过（GFR）、肾小管重吸收 | 利尿药、调钾药 |

### 病理生理学

- **炎症反应**：COX-2 上调 → NSAIDs 抗炎机制
- **免疫病理**：I/II/III/IV 型超敏反应
- **肿瘤病理**：细胞增殖/凋亡失衡，血管生成
- **代谢紊乱**：高血糖、高血脂、离子失衡

## 药物-机体相互作用框架

```
正常生理
    ↓ 病理改变
疾病状态  ←→  药物作用靶点
    ↓              ↓
临床表现    药物效应（治疗 / 不良反应）
```

理解正常生理是预测药物作用和不良反应的前提。例如：
- 理解[[受体]]（GPCR）正常信号传导，才能理解β受体激动剂/拮抗剂的药理作用
- 理解心脏传导系统，才能理解抗心律失常药的作用机制
