# 药物动力学

## 概述

药物动力学（Pharmacokinetics, PK）是研究药物在**体内吸收（Absorption）、分布（Distribution）、代谢（Metabolism）、排泄（Excretion）** 动态过程的学科，简称 **ADME**。是横向连接全部药物、纵向支撑[[处方审核与调剂]] [[用药咨询与治疗管理]] [[慢性病管理]] [[用药安全]] 的核心学科。

---

## ADME 四过程

### 吸收（Absorption）

**给药途径决定吸收特征：**

| 给药途径 | 吸收机制 | 生物利用度（F）| 影响因素 |
|---|---|---|---|
| 静脉注射（IV）| 直接入血 | F = 100%（参照）| 无 |
| 口服（PO）| 被动扩散为主 | 变化大（10%~90%）| 首过效应、pH、食物、胃肠动力 |
| 肌注（IM）| 毛细血管吸收 | 较高 | 注射部位血流 |
| 透皮（TD）| 被动扩散 | 低 | 角质层厚度、药物脂溶性 |
| 吸入（IH）| 肺泡吸收 | 高 | 粒子大小（1~5 μm）|

**达峰时间（Tmax）和峰浓度（Cmax）** 是口服制剂的重要PK参数。

### 分布（Distribution）

**表观分布容积（Vd）** = 体内药量 / 血浆浓度
- Vd 大 → 药物分布广（脂溶性、组织亲和性高）
- Vd 小 → 药物主要停留在血浆

**特殊分布现象：**
- **血脑屏障（BBB）**：只有脂溶性高、分子量小的药物可通过 → [[精神与中枢神经用药]]多需通过BBB
- **胎盘屏障**：几乎所有药物均可通过 → 妊娠期用药安全
- **蛋白结合率**：高蛋白结合率药物可被其他药物置换（如华法林）

### 代谢（Metabolism）

**主要场所：肝脏（CYP450 酶系）**

| CYP酶 | 底物代表 | 诱导剂 | 抑制剂 |
|---|---|---|---|
| CYP3A4 | 大多数药物（50%）| 利福平、糖皮质激素 | 酮康唑、大环内酯 |
| CYP2D6 | β阻滞剂、抗抑郁药 | — | 奎尼丁、氟西汀 |
| CYP2C19 | 奥美拉唑、氯吡格雷 | 利福平 | 奥美拉唑、氟伏沙明 |
| CYP1A2 | 茶碱、咖啡因 | 吸烟、奥美拉唑 | 环丙沙星、西咪替丁 |

**首过效应**：口服药物经肠道吸收后先进入肝门静脉，部分药物在肝脏被代谢（"首过"），降低生物利用度。

### 排泄（Excretion）

- **肾脏排泄**：肾小球滤过（GFR）、肾小管分泌、肾小管重吸收
  - 经肾排泄药物需根据肾功能（Clcr）调整剂量
  - [[泌尿系统用药]]和[[水电解质营养用药]]涉及排泄机制
- **胆汁/粪便排泄**：肝肠循环（部分药物如地高辛经胆汁排泄）
- **肺排泄**：挥发性麻醉药

## 重要PK参数

| 参数 | 含义 | 临床意义 |
|---|---|---|
| **AUC** | 血药浓度-时间曲线下面积 | 生物利用度、暴露量 |
| **Cmax / Tmax** | 峰浓度 / 达峰时间 | 速效制剂评价 |
| **t1/2（半衰期）** | 血药浓度下降50%所需时间 | 给药间隔确定 |
| **CL（清除率）** | 单位时间清除含药血浆体积 | 肝/肾功能评估 |
| **Vd（表观分布容积）** | 药物分布范围 | 负荷剂量计算 |
| **F（生物利用度）** | 口服 vs 静注的AUC比 | 剂型评价 |

**稳态血药浓度（Css）** = (F × 给药剂量的 AUC) / CL

---

## 临床应用

1. **给药方案设计**：根据 t1/2 确定给药间隔
2. **剂量调整**：肾功能不全时减量（如氨基糖苷类）
3. **药物相互作用预测**：CYP450 诱导/抑制
4. **治疗药物监测（TDM）**：地高辛、苯妥英、茶碱等窄治疗指数药物
5. **生物等效性评价**：仿制药一致性评价（AUC、Cmax 90%CI 80%~125%）
