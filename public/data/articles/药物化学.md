# 药物化学

## 概述

药物化学（Medicinal Chemistry）从**化学结构**出发，研究药物的**构效关系（Structure-Activity Relationship, SAR）**，决定药物的**选择性靶点**和**代谢特征**，是连接化学基础与药理学、药物动力学的核心桥梁。

---

## 核心概念

### 构效关系（SAR）

药物的化学结构微小改变可显著影响：
- 与靶点的亲和力（Affinity）
- 药效强度（Potency）
- 选择性（Selectivity）
- 代谢稳定性（Metabolic stability）
- 理化性质（溶解度、脂溶性）

### 药效团（Pharmacophore）

药物分子中与靶点结合的关键结构特征：
- 氢键供体/受体
- 疏水基团
- 电荷中心
- 芳香环体系

### 典型结构类型与代表药物

| 结构类型 | 代表药物 | 作用靶点 |
|---|---|---|
| β-内酰胺环 | 青霉素、头孢菌素 | 细胞壁（PBPs） |
| 喹诺酮母核 | 左氧氟沙星 | DNA旋转酶 |
| 二氢吡啶环 | 硝苯地平、氨氯地平 | 钙通道（L型） |
| 苯二氮䓬环 | 地西泮、氯硝西泮 | GABA-A受体 |
| 吲哚环 | 褪黑素、吲哚美辛 | COX酶、色胺受体 |
| 磺酰胺基 | 氢氯噻嗪、塞来昔布 | CA酶、COX-2 |

### 前药原理（Prodrug）

- **设计目的**：改善溶解度、生物利用度，减少刺激性
- **常见例子**：
  - 伐昔洛韦（Valacyclovir）= 缬氨酸酯前药，口服后在肠道水解为阿昔洛韦
  - 舒林酸（Sulindac）= 可逆性还原为活性硫化物

### 手性药物

- **对映体活性差异**：
  - 相同：普萘洛尔（S/R）
  - 相反：沙利度胺（S = 致畸，R = 镇静）
  - 相差大：华法林（S = R 的 3~5 倍活性）

## 化学结构与药动学的关系

| 化学特征 | 对药动学的影响 |
|---|---|
| 亲脂性↑（LogP↑） | 口服吸收↑、CNS渗透↑、肝代谢↑ |
| 极性↑（氢键↑） | 肾排泄↑、口服吸收↓ |
| 分子量↑ | 口服吸收↓（遵循Lipinski规则，MW < 500） |
| pKa | 影响离子化程度，进而影响吸收和分布 |
